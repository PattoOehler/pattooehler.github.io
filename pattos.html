PattOS

My personal operating system made from scratch in C++

  • Code

 This OS was made to better understand the internal workings of an operating system. It has its own command line as well as its own filesystem. All attached components needed a device driver to work including the Hard Drives, Mouse, Keyboard, and Monitor(all built from scratch).
